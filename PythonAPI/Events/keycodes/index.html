Event Keycodes

In order to see if the user pressed a key, you'll need to use the correct keycodes for it. In cave, you can do this by passing the event type (from cave.event) or by the key name as a string. See cave.Events for more details.

In this page you'll find every key name.

Mouse Events

Event Type Event Name as String Key
cave.event.MOUSE_LEFT "MOUSE_LEFT" Mouse Left
cave.event.MOUSE_MIDDLE "MOUSE_MIDDLE" Mouse Middle
cave.event.MOUSE_RIGHT "MOUSE_RIGHT" Mouse Right
