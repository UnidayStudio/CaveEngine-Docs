engine.TransformComponent

Variables
position : engine.Vector3

scale : engine.Vector3

Note that you can't get/set the rotation using regular variables as you can with the position and scale. That's due to the way the engine works internally. So use the getters and setters for euler angles or quaternions to rotate as you need to. You can see the details bellow. Sample code (as it would be in the update method of a Component):

[...]
def update(self):
    transform = self.entity.getTransform()

    # Modifying the current euler angles by adding 0.1 on the Y axis:
    euler = transform.getEuler()
    euler.y += 0.1
    transform.setEuler(euler)

    # or you can use this to ROTATE the object:
    # (will produce the same result as the approach above)
    transform.rotate(0, 0.1, 0)

[...]
    # Last but no least, you can also do this to SET the euler:
    # (note that this approach will not produce the same result
    # as the two others presented above. This one wil SET the
    # rotation to [0, 0, 0])
    transform.setEuler(0, 0, 0)

engine.LightComponent

You can adjust various light parameters. Here is a sample code that makes the light color pulse using some timers and the python native math.sin() function:

import engine
import math
import random

class PulsingLight(engine.Component):
    def start(self, scene):
        self.speed = 2.0

        # Creating some timers with a random value to sample from
        self.colorX = engine.Timer(random.random())
        self.colorY = engine.Timer(random.random())
        self.colorZ = engine.Timer(random.random())

    def compute(self, value):
        # Maps the timer into a sin curve ranged [0 - 1]
        return (math.sin(value * self.speed) + 1) / 2

    def update(self):       
        light = self.entity.get("Light Component")

        light.color.x = self.compute(self.colorX.get())
        light.color.y = self.compute(self.colorY.get())
        light.color.z = self.compute(self.colorZ.get())

        print(light.color.x, light.color.y, light.color.z)

        # Don't forget to reload the light, otherwise it will not be updated:
        light.reload()


    def end(self, scene):
        pass
